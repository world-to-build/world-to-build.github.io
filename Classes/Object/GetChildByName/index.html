    GetChildByName

    Description

    GetChildByName is a Function on Object
    Gets a child of this Object with the given name, if there is one.

    Example
    local testVariable = Object.GetChildByName("Toaster")
    Usage

    network usage: Any (Client or Server/Host)
    returns: Object
    parameters:
    - name (string)

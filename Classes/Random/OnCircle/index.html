    OnCircle

    Description

    OnCircle is a Function on Random
    Gets a random point on the outside of a circle of a given radius

    Example
    var testVariable = Random.OnCircle(1.0)
    Usage

    network usage: Any (Client or Server/Host)
    returns: Vector2
    parameters:
    - radius (float)

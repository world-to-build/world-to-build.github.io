    GetPlayerByNetId

    Description

    GetPlayerByNetId is a Function
    Returns the player with the matching network ID if they exist in the world.

    Example
    local testVariable = GetPlayerByNetId(1)
    Usage

    network usage: Any (Client or Server/Host)
    returns: Player
    parameters:
    - netId (int)
